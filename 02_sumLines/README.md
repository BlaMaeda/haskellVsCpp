# Description

Read lines from stdin. If the line is a list of integers
separated with spaces, print the sum of them. Otherwise,
print "Wrong line".

# Examples

### Example 1

`input1.txt`

    2 2
    7 7 7 7 7 7
    -1 1 0
    10

    -3 -7

`output1.txt`

    4
    42
    0
    10
    0
    -10
